# ATL — Anytime Learning Platform

A full-stack online learning platform with structured lessons, 
quizzes, coding challenges, and an AI-powered learning roadmap.

## Features
- Student dashboard with lessons, quizzes, and challenges
- Admin panel for content management
- Google OAuth authentication
- AI-powered personalized learning roadmap (Groq API)
- Story mode, skill test, and resource library
- Points and progress tracking system

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Tailwind CSS, Vite |
| Backend | Node.js, Express.js |
| Database | Supabase (PostgreSQL) |
| Auth | Google OAuth via Supabase |
| AI | Groq API |

## Project Structure
- /frontend — React app (student + admin UI)
- /backend — Node.js REST API
