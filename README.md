## 🚀 About

Full-stack online learning platform with structured lessons, quizzes, 
coding challenges, and an AI-powered personalized learning roadmap.

## ✨ Features

- 🎓 Structured programming lessons (HTML, CSS, JS, React, Python, AI/ML)
- 📝 Quizzes with timer and scoring system
- ⚡ Coding challenges
- 📖 Story Mode — learn through narrative chapters
- 🤖 AI Support Bot (Groq API)
- 👑 Admin panel — manage users, lessons, quizzes, resources
- 🔐 Google OAuth authentication
- 📊 Progress tracking + XP points system

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Tailwind CSS, Vite |
| Backend | Node.js, Express.js |
| Database | Supabase (PostgreSQL) |
| Auth | Google OAuth |
| AI | Groq API |
